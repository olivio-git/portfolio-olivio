SDK de autenticación open source publicado en NPM (`sdk-simple-auth`) con **cero dependencias de runtime**, construido para resolver el problema repetitivo de implementar auth en cada proyecto.

Ofrece soporte **multi-backend con auto-detección** (Node/Express, Laravel Sanctum y JWT estándar) y un **refresh inteligente de tokens** con buffer time, retry logic y storage flexible (LocalStorage, IndexedDB, Memory). Se distribuye con triple build (CJS, ESM, UMD), tipos de TypeScript completos y un React hook listo para usar.
